## Getting Started

These instructions will help you in setting up the project and running the game on your local machine for development and testing. 

## Ruby

### Installation

MacOS
```
brew install ruby
ruby -v
```

Ubuntu
```
sudo apt-get install ruby-full
ruby -v
```

### Running

```
ruby ruby.rb
```
